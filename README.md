# 🏰 幻境密室 — 多人在线聊天解谜游戏

> Mystery Room · 多人实时协作，用聊天解开神秘谜题！

---

## 🎮 游戏玩法

1. **输入昵称**，选择头像，进入游戏大厅
2. **创建或加入房间**（2~6人），选择密室主题
3. **房主点击开始游戏**，所有人进入解谜环节
4. 阅读密室场景，**在聊天框讨论**，**提交答案**
5. 答对越快得分越高，全部5关通关后查看排行榜！

### 密室主题
| 主题 | 描述 |
|------|------|
| 🏰 古堡谜案 | 神秘古堡中的悬疑谜题，考验逻辑思维 |
| 🚀 太空迷航 | 宇宙探索主题，需要科学知识 |
| 🌊 海底秘密 | 深海探险，挑战海洋知识 |
| 🎲 随机主题 | 随机选择以上任意主题 |

### 计分规则
- 答对：根据剩余时间计分（最高 100 分，最低 10 分）
- 使用提示：本题最高分 -20 分
- 超时未答对：0 分，系统揭晓答案后进入下一题

## 🚀 快速启动

### 环境要求
- Node.js 16+
- npm

### 安装启动
```bash
# 进入项目目录
cd chat-game

# 安装依赖
npm install

# 启动服务器
npm start
```

### 访问游戏
打开浏览器访问：**http://localhost:3000**

> 💡 **多人测试技巧**：可以同时打开多个浏览器标签，或用不同设备访问同一局域网 IP

## 📁 项目结构

```

├── package.json              # 项目配置
├── README.md                 # 本文件
├── PROJECT_OUTLINE.md        # 详细设计文档
├── server/                   # 后端代码
│   ├── index.js              # 服务器入口 (Express + Socket.io)
│   ├── socketHandler.js      # Socket 事件处理
│   ├── roomManager.js        # 房间管理

    │   ├── global.css        # 全局样式

    └── js/

        ├── index.js          # 首页逻辑
        ├── lobby.js          # 大厅逻辑
        └── room.js           # 房间逻辑
```

## 📝 开发说明

- 游客模式：无需注册，输入昵称即玩
- 房间数据存储在内存中，重启服务器后数据清空
- 最多支持 6 人同时游戏
- 每道题限时 90 秒，共 5 道题
